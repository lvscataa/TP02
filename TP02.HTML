1.1 - ¿Qué es una URL?

R: Una URL es una cadena de texto que especifica donde encontrarse un recurso (video, imagen, documento, etc)

1.2 - ¿Qué es una ruta absoluta?

R: Una ruta absoluta es una dirección web que contiene todas la información necesaria para acceder al recurso

1.3 - ¿Qué es una url relativa?

R: Una URL relativa es una estructura de URL que describe la ruta a un archivo relativo al actual documento.

1.4 - ¿Cual es la diferencia entre una url relativa y una url absoluta?

R: La diferencia de una url relativa y una url absoluta es que la url absoluta contiene todas la información necesaria para acceder al recurso.
En cambio, la url relativa es una estructura que describe la ruta a un archivo relativo al actual documento.

1.6 - ¿Qué es un directorio? ¿qué relación hay con una carpeta?

R: Un directorio es un archivo en el que dentro de este se pueden guardar más archivos ya sean imagenes,documentos,etc.
Una carpeta y un directorio cumplen la misma función ya que en ambos se pueden guardar archivos, documentos de texto, imagenes, etc.

1.7 - En la estructura de un proyecto web, ¿a qué se llama directorio raíz? ¿Con qué símbolo se representa?

R: Se denomina directorio raíz a el primer directorio de una jerarquía. Este mismo contiene todos los subdirectorios y se representa con una barra inclinada "(/)"

1.10 - ¿Cuál es el propósito de la etiqueta anchor?

R: El proposito de la etiqueta "anchor" es crear enlaces a otras páginas, documentos, recursos, etc. Se representa con (a)

1.11 - ¿Qué atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

R: El atributo que utiliza la etiqueta anchor es "href", este atributo proporciona la dirección de la página o recurso (su URL) al navegador.

1.12 - ¿Qué sucede si el atributo "href" en la etiqueta anchor está vacío?

R: Lo que pasa cuando el atributo "href" está vacío es que en la pagina cuando queremos entrar a un enlace especifico no se va a poder porque no tiene una dirección.

1.13 - ¿Cuál es el uso del atributo "target" en la etiqueta anchor y cuáles son los posibles valores que puede tener?

R: El uso que tiene el atributo "target" en la etiqueta anchor es que especifica en donde mostrar la URL enlazada.
Los valores que puede tener son:
Self: Carga la url en el contexto actual.
_blank: carga la URL en un nuevo contexto de navegación.

1.14 - Entonces ¿qué es un hipervínculo?

R: Un hipervínculo o un enlace es una cadena de texto que especifica donde puede encontrarse un recurso.
