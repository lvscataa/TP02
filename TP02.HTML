1.1 - ¿Qué es una URL?

R: Una URL es una cadena de texto que especifica donde encontrarse un recurso (video, imagen, documento, etc)

1.2 - ¿Qué es una ruta absoluta?

R: Una ruta absoluta es una dirección web que contiene todas la información necesaria para acceder al recurso

1.3 - ¿Qué es una url relativa?

R: Una URL relativa es una estructura de URL que describe la ruta a un archivo relativo al actual documento.
